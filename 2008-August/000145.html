Am Samstag, den 16.08.2008, 19:07 +0930 schrieb Jayson Hay:
> Hi there,
> 
> Can a key command using the windows key be used in the Keyboard/Mouse section of 
> a LBRC Profile? What I want to do is control Amarok (without having to jump 
> through their hoops) but to do this I need to send key combos like Win+B
> 
> Is this possible?

Sure - you give the key to be pressed together seperated by a "+" sign.
For example LEFTALT+TAB yields a change of windows. For you this should
be:

LEFTMETA+B (this would be left windows key and b pressed together)
RIGHTMETA+B (this would be right windows key and b pressed together)
COMPOSE+B (this would be menu key and b pressed together)

I don't know whether the keybindings are case sensitive, but to generate
a uppercase b try this: LEFTMETA+LEFTSHIFT+B
