another try ...

I changed the profile we advertise again, this time it announces, that
we use the serial port profile. As my bluetooth receiver announces the
same profile I have some hope to get around the potential "security
measures" of the manufactor.

So - what do you have to do?

1. Grab another build of the j2me client from the usual location
(http://www.doppel-helix.eu/LBRC.ja*)
2. Change BTServer.py: 

def __init__(self, name = "LBRC", serverid = "9c6c8dce-9545-11dc-a3c1-0011d8388a56"):

has to be changed to:

def __init__(self, name = "LBRC", serverid = "0x1101"):

There is little more, that I could imagine that could go wrong ....
