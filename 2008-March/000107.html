Hi there everybody!

I recently stumbled across LBRC and found it very neat tool so my thanks
to everybody created this software. I did not find any configuration
examples for phones having QWERTY keyboards so I though I could share
one I created for Nokia 9300i beacuse it was really pain to create it
(Finnish keyboard layout, probably slight adjustments are needed for
other layouts?). My apologies if this is wrong place to send this!

I found few problems though:

- Pressing * character from mobile (Chr + Å) does cause LBRC client to
disconnect.
- Ctrl+E does quit whole client (though I think this is because JVM and
nothing can be done for it?)
- Pressing Ctrl+I will send the same keycode than pressing tab
- Pressing Ctrl+A/F/Z/X/C/V does not send keycode at all
- Pressing Ctrl+H will send the same keyvode than backspace
- Pressing Ctrl+J will send the same keycode than enter

Otherwise everything works fine. Below is my .lbrc.conf:
