> I have Ubuntu 7.10, How install your program by using SVN ?

Three things you should do first:

     1. remove the debian/ubuntu package (make sure the dependencies
        keep being installed!)
     2. install the subversion utilities (apt-get install subversion)
     3. make sure these packages are there: libdbus-glib-1-dev and gcc

After that you can checkout the sources from the repository by:

svn checkout svn://svn.berlios.de/lbrc/trunk lbrc

This will create the new directory lbrc in which you will find the
latest sources of lbrc. 

Change into the directory and run build_dbus_uinput_bridge. This will
build the uinput bridge. Then you can run LBRC-applet or LBRCdbus
directly from the directory.

Good luck
